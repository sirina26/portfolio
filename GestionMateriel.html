La société « GestionMatos » est une entreprise de maintenance d’équipements informatiques. Elle travaille pour le compte de plusieurs sociétés (ses clients) et ce sur divers sites. GestionMatos désire se doter d’un logiciel lui permettant de gérer son activité d’interventions.

Il s’agit d’un logiciel sécurisé pour y accéder il existe un système de connexion. Il y a aussi une page d’inscription qui permet de créer un nouveau compte où il faut rentrer un login et un mot de passe. Le mot de passe et le login seront ensuite stockés dans la base de données, le mot de passe sera quant à lui crypté avec une fonction de hachage avant d’être stocké dans la base de données.

Vous trouverez ci-dessous le schéma de la base de données, les différentes tables, les entités de chaque table et les liens entre celle-ci permettant de mieux comprendre la gestion des données.

Pour s’inscrire sur l’application l’utilisateur doit tout d’abord rentrer son login. Mais il doit faire attention de ne pas rentrer un login qui existe déjà dans la base de données. Le formulaire d’inscription s’occupera de vérifier si le login existe déjà dans la base et affichera un message d’erreur pour indiquer que le nom d’utilisateur existe déjà.

Dans chaque onglet de l’application on peut choisir de faire trois actions : « Ajouter » c’est-à-dire que l’on peut ajouter des informations soit concernant le client, le site, le matériel nécessaire ou encore le type d’intervention ou des informations sur l’intervention elle-même. « Modifier », cette fonctionnalité permet de modifier des informations déjà rentré dans la base de données afin de corriger les informations précédemment ajouter. « Supprimer » cette dernière fonctionnalité permet quant à elle de supprimer une information. Pour utiliser les fonctionnalités de modification et de suppression il faut cliquer sur l’information voulu dans le tableau à gauche puis soit changer les informations dans le formulaire pour modifier soit cliquer sur supprimer pour les effacer. La modification se fera automatiquement dans la base de données.
